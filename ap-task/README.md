## Getting Started

To run this project, follow these steps:

1. **Clone this repo in your VS Code terminal:**

   ```shell
   git clone <repository_url>

2. **Navigate to the project's root directory:**
   ```
   cd ap-task
3. **Install npm modules:**
   ```
   npm install
4. **Start the project locally::**
   ```
   npm start
